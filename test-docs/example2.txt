React Hooks 使用笔记

useState:
- 用于在函数组件中添加状态
- 返回一个数组：[state, setState]
- 初始值可以是任意类型或函数

useEffect:
- 处理副作用
- 接收两个参数：effect 函数和依赖数组
- 可以返回清理函数
- 依赖数组控制执行时机

useCallback:
- 记忆化回调函数
- 避免子组件不必要的重新渲染
- 配合 React.memo 使用效果更好

useMemo:
- 记忆化计算结果
- 只在依赖项改变时重新计算
- 不要过度使用，有性能开销

常见错误：
1. 忘记添加依赖项
2. 在条件语句中调用 Hooks
3. 直接修改 state 而不是使用 setState
